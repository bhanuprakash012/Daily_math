# Math Quiz Generator 🧮

A simple, randomized Python script designed to generate math addition worksheets. It covers six distinct difficulty levels, ranging from single-digit addition to multi-number series and triple-digit operations.

## ✨ Features

The script generates **5 questions per category** for the following types:
1. **Single Digit:** Standard $x + y$ format.
2. **Single Digit (Series of 5):** Adding five single-digit numbers together.
3. **Double Digit:** Standard $xx + yy$ format.
4. **Double Digit (Series of 5):** Adding five double-digit numbers together.
5. **Mixed Digits:** 3-digit number + 2-digit number ($xxx + yy$).
6. **Triple Digit:** Standard $xxx + yyy$ addition.
